GET

Example request

curl --location --request GET 'https://martinservera-test.pindeliver.com/api/v2_0/Order/get/{ORDER-ID}' \
--header 'X-PINDELIVER-API-KEY: {API-KEY}' \
--header 'X-PINDELIVER-API-CLIENT-KEY: {CLIENT-KEY}' \
--data-raw ''
