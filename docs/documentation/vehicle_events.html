  "id": "{vehicle_id}",
  "url": "https://localhost/api/v2_0/vehicle/get/{vehicle_id}",
    "external_id": "B1"
}

Terminology

Property Description
VEHICLE_WAS_CREATED A vehicle was created
VEHICLE_WAS_DELETED A vehicle was deleted
VEHICLE_WAS_UPDATED A vehicle was updated
